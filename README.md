# cOOlest

![alt text](https://github.com/ramruizni/cOOlest/blob/test/0.gif)

cOOlest is a MP3 player for Android copatible down to sdk 16. Has multiple dynamic themes, playback information on lockscreen, playlist creation, search functionality and audio focus interactions.
